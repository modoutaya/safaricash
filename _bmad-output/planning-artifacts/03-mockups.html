📋 Informations Techniques - SafariCash
Système d'épargne personnelle pour collecteurs
Chaque membre a son compte séparé avec calcul individuel des cycles de 30 jours calendaires.
🎯 Fonctionnalités Core :
  • Gestion de 150+ membres individuels
  • Cotisations journalières avec rattrapage
  • Prêts/avances sans intérêts
  • Calcul automatique des soldes finaux
  • Génération de reçus électroniques
    • 💰 Logique Métier :
  • Cycle = 30 jours calendaires fixes
  • Commission = 1 jour de cotisation
  • Remboursement = Total - Commission - Avances
  • Données séparées par membre
  • Pas de mélange entre comptes
    • 📱 Interface :
  • PWA installable (smartphone/tablette)
  • Design mobile-first optimisé
  • Navigation à 4 onglets principaux
  • Actions rapides (2-3 taps max)
  • Stockage local + synchronisation cloud
    • 🛠️ Workflow Utilisateur :
  • Dashboard → Vue d'ensemble quotidienne
  • Membres → Gestion complète CRUD
  • Transaction → Enregistrement rapide
  • Profil → Détails + historique membre
